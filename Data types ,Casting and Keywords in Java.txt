Q3.What is final class?
Ans:A class declared with final keyword is called final class.A class which is not inherited by another class.

Q4.what are the ways to make a class final?
Ans: First way to make class is final is to declare class with final keyword and another way to make classs final is to make all its constructors private as .If a class has only private subclass it can not be subclassed.

Q5.Can we create an instance of final class in another class ?
